# heart-disease-data-analysis
Exploratory data analysis of a heart disease dataset using Python to identify patterns and factors associated with heart disease.
The analysis focuses on understanding relationships between clinical variables and the presence of heart disease.

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib

## Objectives

* Perform exploratory data analysis (EDA) on the heart disease dataset
* Identify key variables associated with heart disease
* Visualize patterns and relationships within the dataset

## Files in this Repository

* `heart_disease_analysis.ipynb` – Jupyter notebook containing the full data analysis
* `heart_disease.csv` – Dataset used for the analysis

## Key Analysis Steps

* Data loading and inspection
* Data cleaning and preprocessing
* Exploratory data analysis
* Data visualization to identify patterns and trends


Data analysis project developed as part of my learning journey in data analytics using Python.
